# Computer Fundamentals

This module builds the foundation required to understand how computers actually work before learning programming.

---

# Objective

After completing this module, I should be able to understand:

- How a computer works
- How data is represented
- How memory works
- How the CPU executes instructions
- How storage works
- How software communicates with hardware
